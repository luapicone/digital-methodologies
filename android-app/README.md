# Android App

Proyecto Android nativo del MVP antiestafas.

## Stack
- Kotlin
- Jetpack Compose
- Navigation Compose
- Material 3

## Cómo abrirlo
1. Abrir Android Studio.
2. Seleccionar la carpeta `android-app`.
3. Esperar sincronización de Gradle.
4. Ejecutar el módulo `app` en un emulador Android.
